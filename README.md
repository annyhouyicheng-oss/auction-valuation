# 競拍股價即時評估系統

> 台股競拍（承銷競價）三層評估工具，輸入個股基本資料即時產出完整估值報告。

🔗 **線上使用**：`https://<你的帳號>.github.io/<倉庫名稱>/`

---

## 功能說明

### 三層評估邏輯

| 階段 | 內容 |
|------|------|
| 第一層：量化估值 | 以起標價為基準，計算 ×1.2 ～ ×2.0 各錨點，並輔以市價折扣（×0.75/0.80/0.82）及承銷價加減法 |
| 第二層：質化評估 | 依 EPS × 產業本益比（平均／中位數）推算合理股價，並計算目前本益比 |
| 第三層：競拍後檢討 | 填入實際得標均價後，自動推算一般賣出目標（×1.15～1.21）與優質賣出目標（成本×1.6） |

### 競拍篩選規則（排除條件）

以下任一條件成立即建議排除：

1. **日均量不足**：過去6個月日均量 < 150
2. **本益比偏高**：產業本益比均值 > 20
3. **市價灌水**：目前市價 > 起標價 × 1.8
4. **面額過低**：興櫃前面額 < 10 元
5. 上市創新版（4大類，手動判斷）

### 賣出策略（掛牌前5日）

| 類型 | 目標 | 適用情境 |
|------|------|----------|
| 一般出場 | 得標均價 × 1.15 | 正常標的 |
| 一般上限 | 得標均價 × 1.21 | 市場動能佳 |
| 保守出場 | 得標成本 × 1.15 | 風險控制優先 |
| 優質出場 | 得標成本 × 1.60 | 低本益比＋高EPS＋KY股 |

> 備註：下單量少量時，競拍價請集中於金字塔上端前 1/3

## 輸入欄位說明

| 欄位 | 必填 | 說明 |
|------|------|------|
| 股票名稱 | 建議 | 顯示用，不影響計算 |
| 起標價（底價） | **必填** | 競拍底標價，所有錨點基準 |
| 承銷價 | 選填 | 用於承銷加減法估值 |
| 目前市價 | 選填 | 用於市價折扣估值與灌水檢核 |
| EPS（年度） | 選填 | 用於本益比法估價 |
| 產業本益比（平均） | 選填 | 同產業個股本益比平均值 |
| 產業本益比（中位數） | 選填 | 同產業個股本益比中位數 |
| 實際得標均價 | 選填 | 填入後產出賣出策略 |
| 得標成本（含費） | 選填 | 空白則自動計算（得標均價 × 1.05） |

## 快速使用範例

以 **三集瑞-KY** 為例：

| 欄位 | 數值 |
|------|------|
| 起標價 | 364.54 |
| 承銷價 | 433.8 |
| 市價 | 550 |
| EPS | 7.78 |
| 得標均價 | 474 |

估值結果：建議上限 546.81、灌水警戒 656.17、得標均×1.15賣出目標 545.1

## 技術說明

- **純靜態網頁**，無後端、無伺服器，完全在瀏覽器執行
- **無外部 API 相依**，所有計算皆本機完成，數據不會上傳
- 依賴：
  - [Chart.js 4.4.1](https://www.chartjs.org/)（圖表）
  - [Noto Sans TC + DM Mono](https://fonts.google.com/)（字型，需網路）
