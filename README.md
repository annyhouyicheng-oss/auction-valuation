# 競拍股價即時評估系統 v2.0

> 輸入股票代號即自動串接證交所 / 櫃買中心即時資料，填入競拍參數後產出完整估值報告。

🔗 **線上使用**：`https://<你的帳號>.github.io/<倉庫名稱>/`

---

## 功能亮點 v2.0

| 功能 | 說明 |
|------|------|
| 🔍 即時查詢 | 輸入股票代號，自動抓取即時市價、本益比、EPS |
| 📊 日均量計算 | 自動抓取近6個月歷史資料，計算日均量並判斷是否符合篩選條件 |
| 🏷️ 自動帶入 | 市價 / EPS 自動填入計算欄位，可手動覆蓋 |
| 📈 完整估值 | 起標價 ×1.2～×2.0 全套錨點 + 市價折扣 + 承銷加減法 |
| 💰 賣出策略 | 填入得標均價後，自動計算4種出場目標 |
| 📉 圖表視覺化 | 所有錨點 + 即時市價 + 得標均價同圖比較 |
| 🌙 深色模式 | 自動跟隨系統外觀 |

## 資料來源

| 資料 | 來源 | 備註 |
|------|------|------|
| 即時市價、本益比、EPS | mis.twse.com.tw | TSE + OTC 皆支援 |
| 月歷史成交（計算日均量） | twse.com.tw / tpex.org.tw | 自動判斷上市/上櫃 |

所有 API 皆為公開免費資料，無需申請金鑰。

---

## 三層評估邏輯

**第一層 — 量化錨點**：起標價 ×1.2/1.3/1.4/1.5(上限)/1.8(灌水)/2.0(絕對)，以及市價×0.75/0.80/0.82 折扣法

**第二層 — 本益比估值**：EPS × 產業平均/中位數本益比

**第三層 — 賣出策略**：得標均價×1.15/1.21，得標成本×1.15/1.60（優質標的）

---

## 篩選排除條件（自動判斷）

1. 日均量 < 150 張（系統自動計算6個月均值）
2. 產業本益比均值 > 20
3. 市價 > 起標×1.8（灌水，系統自動標示）
4. 興櫃前面額 < 10 元
